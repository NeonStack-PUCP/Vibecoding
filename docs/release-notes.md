# Release Notes - ReportaP'

## Prototipo de arquitectura - ReportaP'

Entrega correspondiente al Hito 1 / Sprint 0.

## Objetivo del release

Este release documenta el prototipo inicial de arquitectura de ReportaP', una plataforma cívica para registrar denuncias y propuestas ciudadanas geolocalizadas, cruzarlas con datos públicos del Estado peruano y generar expedientes formales con apoyo de IA.

## Contenido incluido

- README inicial del proyecto.
- Documentación de arquitectura del sistema.
- Diagrama de módulos principales.
- Modelo de datos inicial.
- Diagrama de despliegue propuesto.
- Documentación del uso de herramientas de IA.
- Estructura inicial del repositorio.

## Estado del proyecto

El proyecto se encuentra en etapa de prototipo de arquitectura.

En este hito no se implementan funcionalidades completas. El objetivo es dejar documentada la estructura técnica inicial, los módulos principales, las herramientas propuestas y la comunicación entre componentes.
